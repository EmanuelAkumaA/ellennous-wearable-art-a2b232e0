
## Plano: Conectar ao GitHub

Vou conectar o projeto ao GitHub usando o connector disponível.

### Passos
1. Verificar conexões disponíveis no workspace
2. Executar `standard_connectors--connect` com `connector_id: "github"`
3. O usuário será guiado para autorizar e selecionar/criar o repositório

### Resultado esperado
- Projeto sincronizado bidirecionalmente com GitHub
- Commits automáticos em ambas direções
- Código disponível para download/clone
